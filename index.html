O que é o Git?

É um software de controle de versões.

Por que usar o Git?

  • Versionamento;
  • Controle;
  • Organização;
  • Integração Contínua.

Um ponto importante é que não se deve commitar código que não funciona, ou que esteja com problemas.
Os commits devem ser feitos a cada alterção significativa.

O que é o GitHub?

É uma plataforma de rede social para programadores.

Por que usar o GitHub?

  • Repositórios ilimitados;
  • Hospedagem de código-fonte;
  • Características de rede social;
  • GitHub pages integrado;
  • Colaboração;
  • Forks.
